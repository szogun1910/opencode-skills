# LLM Wiki Plugin

Generowanie dokumentacji z kodu przy użyciu LLM dla OpenCode.

**GitHub:** https://github.com/praneybehl/llm-wiki-plugin

## Funkcje

- **Generowanie dokumentacji z kodu** - automatyczne tworzenie docs z source
- **Wiki oparte na AI** - inteligentna organizacja wiedzy projektowej
- **Auto-dokumentacja** - dokumentuj nowe pliki automatycznie

## Struktura

```
skills/         # Skille dla pluginu
commands/       # Komendy CLI
```

## Użycie

1. Plugin analizuje kod źródłowy
2. Generuje dokumentację używając LLM
3. Organizuje wiedzę w formacie wiki
4. Utrzymuje aktualność przy zmianach

## Konfiguracja

Brak dodatkowej konfiguracji - działa out of the box.
